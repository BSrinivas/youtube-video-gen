# YouTube Video Generator - Complete Free Solution

A comprehensive Python application for creating professional YouTube videos using 100% free cloud services and tools.

## 🎯 Overview

This project provides a complete workflow for generating YouTube videos without spending any money. It includes:

- **Script Generation Templates** - AI-powered script templates
- **Asset Management** - Organize videos, images, and audio
- **Cloud API Integration** - Connect to free stock services
- **Project Orchestration** - Complete workflow guidance
- **Best Practices** - Professional video creation guidelines
- **CLI Interface** - Easy command-line management

## ✨ Features

### 1. **Video Project Management**
```python
config = VideoConfig(
    title="My YouTube Video",
    topic="Video creation tips",
    duration=600,  # 10 minutes
    style="educational",

)

project = VideoProject(config)
project.initialize_project()
```

### 4. **Asset Management**
```python
# Add video asset
project.assets.add_video_asset(
    "clip.mp4",
    "https://example.com/video.mp4",
    {"duration": 30}
)

# Add image asset
project.assets.add_image_asset(
    "thumbnail.jpg",
    "https://example.com/image.jpg"
)

# Add audio/music
project.assets.add_audio_asset(
    "music.mp3",
    "https://example.com/music.mp3",
    duration=120
)
```

### 5. **API Integration**
```python
from cloud_apis import CloudAPIManager

manager = CloudAPIManager()

# Search stock videos
videos = manager.search_stock_videos("nature", service="pixabay")

# Search stock images
images = manager.search_stock_images("sunset", service="pexels")

# Search audio
audio = manager.search_audio("upbeat music", service="freesound")

# Generate voiceover
voice = manager.generate_voiceover(
    "Hello world!",
    service="google_tts"
)
```

## 🚀 Quick Start

### Installation

1. **Clone or download the project:**
```bash
# Assuming files are in current directory
cd /path/to/youtube-video-generator
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Set up environment variables (optional):**
```bash
# Create .env file
echo "PEXELS_API_KEY=your_key" > .env
echo "PIXABAY_API_KEY=your_key" >> .env
echo "FREESOUND_API_KEY=your_key" >> .env
echo "ELEVENLABS_API_KEY=your_key" >> .env
echo "GOOGLE_API_KEY=your_key" >> .env
```

### Usage

#### Option 1: Using the CLI
```bash
python youtube_cli.py
```

This opens an interactive menu where you can:
- Create new projects
- Generate scripts
- Search for assets
- View workflows
- Check best practices

#### Option 2: Using the Core Module
```python
from youtube_generator import VideoConfig, VideoProject

# Create configuration
config = VideoConfig(
    title="How to Make YouTube Videos",
    topic="YouTube video creation",
    description="Learn to create videos with free tools",
    duration=600,
    style="educational",
    music_vibe="upbeat"
)

# Create project
project = VideoProject(config)
project.initialize_project()

# Generate script
print(project.script.generate_script_template())

# Export summary
project.export_project_summary()
```

#### Option 3: Using Cloud APIs
```python
from cloud_apis import CloudAPIManager

manager = CloudAPIManager()

# Search across services
results = manager.search_all_services(
    query="nature documentary",
    media_type="video"
)

# Generate voiceover
audio = manager.generate_voiceover(
    "Welcome to my channel!",
    service="google_tts"
)
```

## 🔧 Configuration

### VideoConfig Options

```python
VideoConfig(
    title: str,              # Video title
    topic: str,              # Main topic
    description: str,        # Video description
    duration: int,           # Duration in seconds
    style: str,              # cinematic|casual|educational|energetic
    music_vibe: str,         # upbeat|calm|dramatic|ambient
    include_voiceover: bool, # Include voiceover?
    output_dir: str          # Output directory
)
```

### Style Guidelines

**Cinematic:**
- Slow pans and camera movements
- Dramatic lighting
- Professional color grading
- Orchestral or dramatic music

**Casual:**
- Quick cuts
- Energetic pacing
- Authentic, personal feel
- Modern pop or hip-hop music

**Educational:**
- Clear explanations
- On-screen text and diagrams
- Step-by-step flows
- Calm, focused music

**Energetic:**
- Fast cuts and transitions
- Dynamic camera movements
- Quick pacing
- Electronic or upbeat music

## 📊 Best Practices

### Engagement
- ✓ Hook viewers in first 3 seconds
- ✓ Keep videos 8-15 minutes optimal
- ✓ Add captions/subtitles
- ✓ Use dynamic sound design
- ✓ Cut dead air
- ✓ Maintain visual consistency

### SEO Optimization
- ✓ Use keywords in title
- ✓ Write detailed descriptions (200+ chars)
- ✓ Use relevant hashtags and tags
- ✓ Create custom thumbnails
- ✓ Optimize for click-through rate

### Common Mistakes to Avoid
- ✗ Poor audio quality
- ✗ Inconsistent pacing
- ✗ No clear call-to-action
- ✗ Ignoring analytics
- ✗ Uploading at random times
- ✗ Not engaging with comments

### Monetization Requirements
- YouTube Partner Program (10k subs, 4k watch hours)
- AdSense account setup
- Consistent quality content
- Community engagement
- Regular upload schedule
